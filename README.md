基于 SpringBoot + Vue 的高校资助管理平台。集成奖助贷申请、多级联动评审、资金精准发放及数据可视化看板。支持管理员、导师、学生三端交互，提供完整的教育资助数字化解决方案。

### 大学生资助管理系统

管理员：专业管理、班级管理、学生管理、导师管理、资助项目、项目发布、申请总览、评审记录、评审任务分配、最终审核、资助政策、公告公示、资金发放、学生账户管理、经济情况、证书资料、数据统计

学生：个人信息维护、银行账户、经济情况、证书资料、资助申请、系统消息、在线沟通

导师：个人信息维护、学生档案查看、资助政策查询、待办任务、资助初审、系统消息、在线沟通
